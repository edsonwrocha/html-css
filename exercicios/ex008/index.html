Tipos de formatação de texto

Negrito/Destaque

Parte desse texto está em negrito usando a TAG STRONG(destaque)

Itálico ou Ênfase

Parte desde texto está em itálico usando a TAG EM

Texto Marcado

É possível marcar uma parte do texto usando a tag MARK

É possível adicionar um parâmetro de CSS para modificar a cor do marca-texto, usando o parâmetro "style, background-color e selecionando uma cor, ao adicionar isto na parte head do código ele vai ser usado em todo o texto, já usado dentro da tag só mudará aquela parte do texto

Texto grande e pequeno

Para formatar parte do texto como grande, utilizar tag BIG (Não semântica e obsoleta) ou a tag SMALL(semântica)

Texto excluído

Para mostrar parte do texto, porém informar que ele deve ser desconsiderado, utilizar a tag DEL

Texto inserido

Para mostrar que parte do texto foi colocada posteriormente, e merece ênfase, usar a tag INS

Texto sobrescrito

Para mostrar que parte do texto na parte superior, usar a tag SUB, exemplo: b2 - 4ac

Texto subescrito

Para mostrar que parte do texto foi subescrito, usar tag sub, exemplo: H2O
